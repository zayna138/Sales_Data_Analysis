# Sales Data Analysis

## Project Overview
This project analyzes sales data using Python, Pandas, and Matplotlib to identify sales trends, product performance, and business insights.

## Features
- Sales data analysis
- Product performance tracking
- Highest-selling product identification
- Profit analysis
- Data visualization using charts

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Output
The project generates insights and visual charts from sales data to help understand business performance.
